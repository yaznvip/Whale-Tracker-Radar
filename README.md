# 🐳 Whale Tracker Radar (Ethereum Mainnet)

A professional, real-time blockchain monitoring dashboard designed to track high-value transactions and balances of "Whale" wallets on the Ethereum network.

## 🚀 Live Demo
Check out the live application here: [https://yaznvip.github.io/Whale-Tracker-Radar/](https://yaznvip.github.io/Whale-Tracker-Radar/)

## ✨ Key Features
- **Real-time On-chain Tracking:** Automatically scans the Ethereum blockchain every 15 seconds.
- **Multi-Asset Support:** Monitors **ETH** and major ERC-20 tokens like **USDT**, **USDC**, and **PEPE**.
- **Visual Alerts:** Features a dynamic "Movement Alert" system that highlights assets in red when a balance change is detected.
- **Activity Log:** A live scrolling feed that timestamps every detected movement for precise auditing.
- **No-Key Setup:** Built using public RPC nodes, making it accessible and lightweight.

## 🛠️ Technical Architecture
- **Web3 Integration:** Powered by `Ethers.js` for direct blockchain interaction.
- **Frontend:** Crafted with `HTML5` and `Tailwind CSS` for a sleek, responsive dark-mode interface.
- **Data Source:** Connects to Ethereum Mainnet via stable decentralized RPC providers.

## 📂 Project Structure
- `index.html`: The main interactive dashboard (Frontend).
- `tracker.js`: The CLI version for terminal-based monitoring (Backend logic).
